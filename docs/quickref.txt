ABS( result = ABS(n) : ABS() returns the absolute value of n.
ADDRESS ADDRESS <identifier>[,..] : ADDRESS declares and initialises one or more variables of type address (synonymous with LONGINT).
ALLOC( address = ALLOC(<bytes>[,<memory-type>]) : ALLOC() allocates memory. Memory-type: 0=CHIP, 1=FAST, 2=PUBLIC, 3-5=cleared versions, 6=ANY, 7=ANY cleared (default). Returns 0 on failure. Memory is freed at program exit.
AND AND is the boolean AND operator.
AREA AREA [STEP](x,y) : AREA defines a point for a polygon to be filled by AREAFILL. Up to 20 points.
AREAFILL AREAFILL [mode] : AREAFILL fills a polygon defined by AREA. Mode 0=foreground color (default), 1=inverse color.
ARG$( argument$ = ARG$(n) : ARG$() returns the nth command line argument as a string. ARG$(0) returns the command name. CLI/Shell launched programs only.
ARGCOUNT count = ARGCOUNT : ARGCOUNT returns the number of command line arguments.
ASC( ascii_code = ASC(X$) : ASC() returns the ASCII code of the first character in X$.
ASSEM ASSEM .. END ASSEM : ASSEM allows inline inclusion of assembly source code into the generated A68K file.
ASSERT ASSERT <condition> [, <message>] : ASSERT tests the condition and halts program execution if false, printing "ASSERT FAILED" with an optional message. Exits cleanly.
ATN( arctan = ATN(n) : ATN() returns the arctangent of n.
BACK BACK n : BACK moves the turtle back n steps.
BEEP BEEP : BEEP issues a brief pulse from the speaker to alert the user.
BEVELBOX BEVELBOX (x1,y1)-(x2,y2),style : BEVELBOX renders a Wb 2.x/3.0 style 3D bevel-box. Style: 1=RAISED, 2=RECESSED, 3=STRING-GADGET STYLE.
BIN$( binary$ = BIN$(n) : BIN$() returns a string containing the binary representation of n.
BIND( closure& = BIND(@<sub-name>, <expr1>[, <expr2>..]) : BIND() creates a closure that pre-binds arguments to a SUB. Returns address for use with INVOKE.
BLOCK BLOCK .. END BLOCK : BLOCK permits multiple statements where one is normally expected, such as with IF..THEN..ELSE or CASE.
BREAK BREAK ON|OFF|STOP : BREAK enables, disables or suspends ON BREAK event trapping.
CALL CALL <subprogram>[(params)] : CALL passes control to a user-defined subprogram, shared library function, external function, or machine code routine. Calls can be recursive.
CASE CASE <expression> : <statement> .. END CASE : CASE evaluates expressions as booleans until one is true, then executes the corresponding statement.
CHDIR CHDIR <dirname> : CHDIR changes the current directory.
CHR$( char$ = CHR$(n) : CHR$() returns a string consisting of a single character with ASCII value n.
CHIPSET( chipset = CHIPSET : CHIPSET returns the Amiga's graphics chipset: 0=OCS, 1=ECS, 2=AGA. Use to detect AGA before using screen modes 7-12.
CINT( integer = CINT(n) : CINT() converts n to a signed short integer by rounding the fractional portion.
CIRCLE CIRCLE [STEP](x,y),radius[,color[,start,end[,aspect[,F]]]] : CIRCLE draws a circle or ellipse. Angles in degrees. Trailing F fills the circle.
CLEAR CLEAR ALLOC : CLEAR ALLOC frees all memory allocated by calls to ALLOC. Subsequent use of ALLOC is permitted.
CLNG( longint = CLNG(n) : CLNG() converts n to a signed long integer by rounding the fractional portion.
CLOSE CLOSE [#]<fileNum>[,[#]<fileNum>..] : CLOSE closes one or more open files. At least one file number must be specified.
CLS CLS : CLS clears the current output window or screen and sets the pen position to the upper left corner.
COLOR COLOR fgnd[,bgnd] : COLOR changes the foreground and optionally the background color.
COMMON COMMON [<type>] <identifier> : COMMON declares a static variable with global scope across modules. Type can be ADDRESS, LONGINT, SHORTINT, SINGLE, STRING.
CONST CONST <ident> = [+|-]<constant>[,..] : CONST defines one or more named numeric constants. All constant definitions are global.
COS( cosine = COS(n) : COS() returns the cosine of n.
CSNG( single = CSNG(n) : CSNG() converts n to a single-precision value.
CSRLIN line = CSRLIN : CSRLIN returns the print line in the current user-defined screen or window. Returns 0 in CLI/shell.
CSTR( string$ = CSTR(<address>) : CSTR() coerces a long integer address into a string. The memory block must be NULL terminated.
DATA DATA [constant][,..] : DATA stores numeric and/or string constants into a global data list to be accessed by READ.
DATE$ date$ = DATE$ : DATE$ returns the current system date as mm-dd-yyyy.
DAY day = DAY : DAY returns the day of the week as 0..6 (0=Sunday, 6=Saturday). Reflects the last call to DATE$.
DECLARE DECLARE FUNCTION|SUB|STRUCT .. : DECLARE declares shared library functions, external functions, forward SUB declarations, or structure variables.
DEF DEF [FN]name[(params)] = <expr> : DEF creates a user-defined function. Parameters are passed by value. The FN prefix is optional.
DEFINT DEFINT <letter>[-<letter>][,..] : DEFINT declares all variables starting with specified letters as short integers unless overridden.
DEFLNG DEFLNG <letter>[-<letter>][,..] : DEFLNG declares all variables starting with specified letters as long integers unless overridden.
DEFSNG DEFSNG <letter>[-<letter>][,..] : DEFSNG declares all variables starting with specified letters as single-precision unless overridden.
DEFDBL DEFDBL <letter>[-<letter>][,..] : DEFDBL currently defaults to single-precision since double-precision is not yet supported.
DEFSTR DEFSTR <letter>[-<letter>][,..] : DEFSTR declares all variables starting with specified letters as strings unless overridden.
DIM DIM [<type>]<name>(<i>[,..])[SIZE <n>][ADDRESS <addr>][,..] : DIM creates arrays. Type can be ADDRESS, LONGINT, SHORTINT, SINGLE, STRING. SIZE sets string element length. ADDRESS specifies memory location.
END END : END terminates program execution, closes libraries and performs cleanup.
EOF( result = EOF(n) : EOF() returns -1 if end-of-file reached for file number n, 0 otherwise. Returns -1 if file doesn't exist or isn't open.
EQV EQV is the boolean equivalence operator.
ERR errorcode = ERR : ERR returns the error code from the last failed operation (or 0) and immediately clears the error code.
ERROR ERROR ON|OFF|STOP : ERROR enables, disables or suspends ON ERROR event trapping.
EXIT EXIT FOR : EXIT FOR allows premature exit from a FOR..NEXT loop with proper stack cleanup.
EXP( result = EXP(n) : EXP() returns e to the power n (e = 2.7182818284590).
EXTERNAL EXTERNAL [FUNCTION] [<type>] <identifier> : EXTERNAL declares an external function or variable for cross-module access.
FILEBOX$( path$ = FILEBOX$(title[,default-dir]) : FILEBOX$() invokes a file requester and returns the user's selection as a fully qualified path.
FILES FILES [TO <storefile>][,<target>] : FILES gives a directory listing. Optional storefile captures output. Target can be file, directory or device.
FIX( integer = FIX(n) : FIX() returns the truncated integer portion of n. Equivalent to SGN(n)*INT(ABS(n)).
FONT FONT <name>,<size> : FONT changes the font for the current output window. Works only for WINDOW-created windows.
FOR FOR <var>=x TO y [STEP z] : FOR starts a FOR..NEXT loop. The loop variable is incremented by z (default 1) until it reaches y.
FORWARD FORWARD n : FORWARD moves the turtle forward n steps.
FRE( freemem = FRE(n) : FRE() returns free system memory. n: -1=total, 0=CHIP, 1=FAST, 2=largest CHIP block, 3=largest FAST block.
GADGET GADGET id,status[,gadval,rectangle,type[,style]] : GADGET creates or modifies a gadget. Type: 1=BUTTON, 2=STRING, 3=LONGINT, 4=POTX, 5=POTY.
GADGET( info = GADGET(n) : GADGET() returns gadget info. n: 0=event occurred, 1=gadget number, 2=string/longint value, 3=slider position, 4=gadget address.
GET GET [#]<fileNum>,<structVar>[,<recordNum>] : GET retrieves a record from a random file and stores it in a structure variable.
GLOBAL GLOBAL [<type>] <identifier> : GLOBAL declares a static variable with module scope. Type can be ADDRESS, LONGINT, SHORTINT, SINGLE, STRING.
GOSUB GOSUB <label> : GOSUB transfers control to the specified label. RETURN passes control back to the statement after GOSUB.
GOTO GOTO <label> : GOTO transfers control to the specified label or line.
HANDLE( handle = HANDLE(n) : HANDLE() returns the dos file handle pointer for file number n. Returns 0 if file doesn't exist.
HEADING heading = HEADING : HEADING returns the turtle's current heading in degrees (0..359).
HEX$( hex$ = HEX$(n) : HEX$() returns a string containing the hexadecimal representation of n.
HOME HOME : HOME moves the turtle to its home position.
IF IF <condition> [THEN] .. [ELSE ..] [END IF] : IF starts a conditional block. Supports single-line IF..THEN..ELSE and multi-line IF..THEN..ELSE..END IF.
IFF( info = IFF(channel,n) : IFF() returns IFF file info. n: 0=form name address, 1=width, 2=height, 3=depth, 4=screen mode.
IMP IMP is the boolean implication operator.
INKEY$ key$ = INKEY$ : INKEY$ returns a single character string when a keystroke is pending, otherwise the NULL string.
INPUT INPUT [<prompt>][;|,] var[,..] : INPUT reads values from the keyboard. A semicolon shows "? " prompt, a comma shows nothing.
INPUTBOX( value& = INPUTBOX(prompt[,title][,default][,xpos][,ypos]) : INPUTBOX() invokes a requester for integer input with OK and Cancel gadgets.
INPUTBOX$( value$ = INPUTBOX$(prompt[,title][,default][,xpos][,ypos]) : INPUTBOX$() invokes a requester for string input with OK and Cancel gadgets.
INPUT$( data$ = INPUT$(X,[#]filenum) : INPUT$() returns a string of X characters from the specified file. 32K upper limit for X.
INSTR( pos = INSTR([I,]X$,Y$) : INSTR() searches for Y$ in X$ starting from optional position I. Returns position 1..N or 0 if not found.
INT( integer = INT(n) : INT() returns the largest integer less than or equal to n.
INVOKE INVOKE <variable>([<args>]) : INVOKE performs an indirect call through a function pointer or closure. Can be used as statement or in expression.
KILL KILL <filespec> : KILL deletes a file or directory.
LEFT$( left$ = LEFT$(X$,I) : LEFT$() returns the leftmost I characters of X$.
LEN( length = LEN(X$) : LEN() returns the number of characters in X$.
LET [LET] <variable> = <expression> : LET assigns a value to a variable. The LET keyword is optional.
LIBRARY LIBRARY [CLOSE] [<libname>] : LIBRARY opens or closes Amiga shared libraries. LIBRARY CLOSE closes all or a specified library.
LINE LINE [STEP](x1,y1)[-(x2,y2)[,[color],[b[f]]]] : LINE draws a line or rectangle. b=box, bf=filled box.
LOCATE LOCATE line[,column] : LOCATE changes the print position for the current screen or window.
LOC( position = LOC(n) : LOC() returns the current reading/writing position in the file numbered n.
LOF( size = LOF(n) : LOF() returns the length of the file numbered n in bytes. Returns 0 if file not open.
LOG( logarithm = LOG(n) : LOG() returns the natural logarithm of n. Argument must be greater than zero.
LONGINT LONGINT <identifier>[,..] : LONGINT declares and initialises long integer variables, or LONGINT(X$) converts a string to a long integer.
LONGINT( value& = LONGINT(X$) : LONGINT() returns the numeric value of X$ as a long integer. Supports &H (hex) and &O (octal) prefixes.
MENU MENU menu-id,item-id,state[,title[,command-key]] : MENU creates or modifies menus. State: 0=disabled, 1=enabled, 2=checkmarked.
MENU( info = MENU(n) : MENU() returns menu info. n: 0=menu number, 1=item number, 2=subitem number.
MID$( mid$ = MID$(X$,I[,J]) : MID$() returns J characters from X$ starting at position I. If J is omitted, returns all characters from I to end.
MOD MOD is the modulo arithmetic operator. Example: 101 MOD 10 = 1.
MOUSE( info = MOUSE(n) : MOUSE() returns mouse status. n: 0=left button pressed (-1/0), 1=X position, 2=Y position.
MSGBOX( result = MSGBOX(message,button1[,button2]) : MSGBOX() invokes a system requester with one or two buttons. Returns -1 for first button, 0 for second.
NAME NAME <filespec1> AS <filespec2> : NAME renames a file or directory.
NEXT NEXT [<variable>] : NEXT terminates a FOR..NEXT loop.
NOT NOT is the boolean NOT operator.
OCT$( octal$ = OCT$(n) : OCT$() returns the octal string representation of n.
ON ON <integer-expr> GOTO|GOSUB <label>[,..] : ON branches to one of several labels based on the integer expression value.
OPEN OPEN mode,[#]<fileNum>,<filespec> : OPEN opens a file. Mode: "I"=input, "O"=output, "A"=append, "R"=random access.
OPTION OPTION <switch>+|-[,..] : OPTION sets compiler directives. Switches: 2,b,c,E,i,l,m,O,w. + activates, - deactivates.
OR OR is the boolean OR operator.
PAINT PAINT (x,y)[,color-id[,border-id]] : PAINT flood-fills an enclosed region with the specified color.
PALETTE PALETTE color-id,R,G,B : PALETTE changes colors in the current screen. R,G,B are in the range 0..1.
PATTERN PATTERN [line-pattern][,area-pattern] | RESTORE : PATTERN sets line and/or area draw patterns. PATTERN RESTORE resets to defaults.
PEEK( value = PEEK(<address>) : PEEK() returns an 8-bit value from the specified memory address. PEEKW for 16-bit, PEEKL for 32-bit.
PEEKW( value = PEEKW(<address>) : PEEKW() returns a 16-bit value from the specified memory address.
PEEKL( value = PEEKL(<address>) : PEEKL() returns a 32-bit value from the specified memory address.
PENDOWN PENDOWN : PENDOWN lowers the turtle's pen, enabling drawing by turtle graphics commands.
PENUP PENUP : PENUP raises the turtle's pen, disabling drawing by turtle graphics commands.
POINT( color = POINT(x,y) : POINT() returns the color-id of a point in the current output window or screen.
POKE POKE <address>,<value> : POKE writes an 8-bit value to the specified memory address. POKEW for 16-bit, POKEL for 32-bit.
POKEW POKEW <address>,<value> : POKEW writes a 16-bit value to the specified memory address.
POKEL POKEL <address>,<value> : POKEL writes a 32-bit value to the specified memory address.
POS column = POS : POS returns the print column in the current user-defined screen or window. Returns 0 in CLI/shell.
POTX( value = POTX(n) : POTX() returns the potentiometer reading on pin 5 of game port n (0 or 1).
POTY( value = POTY(n) : POTY() returns the potentiometer reading on pin 9 of game port n (0 or 1).
PRINT PRINT [<expression>][,|;|..] : PRINT prints to the current output window. Semicolon suppresses linefeed, comma adds a tab. ? can be used instead of PRINT.
PRINTS PRINTS [<expression>][,|;|..] : PRINTS prints to an open screen or Intuition window at the current x,y location. Faster than PRINT but not for CLI/shell.
PSET PSET [STEP](x,y)[,color-id] : PSET plots a point in the current output window or screen.
PTAB( PRINT PTAB(n) : PTAB() moves the horizontal print position to the nth pixel in the current output window.
PUT PUT [#]<fileNum>,<structVar>[,<recordNum>] : PUT stores a structure into the next or specified record of a random file.
RANDOMIZE RANDOMIZE <expression> : RANDOMIZE seeds the random number generator. RANDOMIZE TIMER is the most common usage.
READ READ <variable>[,..] : READ assigns the next value(s) from DATA statements to the specified variable(s).
REM REM <comment> : REM starts a single-line comment. Can be substituted by an apostrophe ('). Block comments use {..}.
REPEAT REPEAT .. UNTIL <condition> : REPEAT starts a loop that executes until the condition is true. The loop body executes at least once.

SHORTINT SHORTINT <identifier>[,..] : SHORTINT declares and initialises one or more short integer variables.
